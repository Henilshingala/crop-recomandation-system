# Development Scripts

This folder contains one-off development, data generation, and migration scripts.
These scripts are NOT used in the production Django application, but are kept here for historical reference or future utility.

## Scripts Included:
- `1.py`, `fqgenerator.py`: Initial data generation scripts.
- `analyze_ai_json.py`, `append_ai_json.py`, `append_refusals.py`: Used for expanding the AI FAQ dataset.
- `check_hindi.py`, `debug_langs.py`: Language debugging tools.
- `translate_qa.py`, `translate_qna.py`, `translate_schemes.py`, `update_locales.py`: Translation utilities for multilingual support.
- `test_similarity.py`: Script to test the text similarity algorithms.
